Git API

The purpose of this project is to create an example API server/client which uses Git as the frontend/backend storage, and as the comms protocol.

There will be one "data" repo for the project, which the server will create and clients will clone.

Resources will be stored as .json files in a folder structure matching the URI structure of a restful API

A js git implementation will be used on the front-end application, with an in-browser filesystem for resource storage.
